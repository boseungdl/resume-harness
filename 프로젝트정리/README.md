## 자료 처리 메모
- 노션 zip은 한글 파일명이 깨져 있어 인코딩(cp949/utf-8) 복구 후 재추출 → 각 프로젝트 폴더의 `_노션추출/`(및 VDI는 `_노션추출_clean/`).
- `smartfarm/스마트팜_노션.zip`은 실제로 **충돌주의보(지게차) 팀 노션** → 지게차 정리에 활용.
- 팬택 산출물 zip(626MB, 바이너리)은 파일 목록만 SI 산출물 근거로 사용.
- 이미지형 PDF(광고레이크 강의·logland 발표)는 PyMuPDF 렌더 후 시각 판독.
- 정량 수치는 각 `정리.md`에서 **측정/추정/미측정**을 구분 표기함.
