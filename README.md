# resume-harness

채용 공고를 분석해 지원자 프로필을 근거로 한국어 자기소개서(자소서)를 작성하는 **하네스**. 공고가 없으면 직무 분석으로 범용 자소서를 쓴다. Claude Code 스킬 + 서브에이전트로 구현했다.

## 하네스란

모델(Claude)을 제외한, 모델을 감싸 자소서 작성을 신뢰성 있게 굴러가게 만드는 결정론적 제어층. 분석 → 매칭 → 작성의 오케스트레이션, 자수 검증, 중간 산출물의 상태 관리를 담당한다. *"모델이 아닌 모든 것이 하네스다."*

## 파이프라인

```
공고 → ① 분석 → ② 매칭(사람 검토) → ③ 문항별 작성(병렬) → ④ 검증
```

- **① 분석**: 공고에서 요구역량·인재상·키워드를 뽑고 지원 직무를 추론(사용자 승인)
- **② 매칭**: 요구역량 ↔ 프로필 경험을 표로 연결 — 작성 전 사람이 검토·수정하는 지점
- **③ 작성**: 문항 하나당 에이전트 하나로 병렬 작성. 합니다체·두괄식·STAR, 자수 제한 준수
- **④ 검증**: 자수·문항 적합성·진부 표현·중복 점검

## 사용

```
/write-jaso
```

공고 본문을 붙여넣고 문항·자수를 주면 실행된다. 지원 건마다 `.scratch/<slug>/`에 산출물(posting·analysis·matching·자소서)이 쌓인다.
